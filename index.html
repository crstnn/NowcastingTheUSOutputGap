Nowcasting the Output Gap

Latest update:

The nowcast of the United States output gap is based on the method described in Berger, Morley, and Wong (forthcoming, Journal of Econometrics). Please cite the paper when referring to the nowcast estimate.

The nowcast for the current quarter is reported in the figure using an orange line from the actual estimated output gap for previous quarters. The actual estimated output gap is based on parameter estimates for the mixed-frequency Bayesian VAR specified in Berger, Morley, and Wong and data up to the latest quarter for which real GDP has been released from FRED.

Please see Berger, Morley, and Wong for more details and please cite the paper when using the method or the results presented here.

Below are the three data files necessary to run the code (as seen in the paper).
These data are mainly gathered from FRED but for those time-series that are not timely, they are scraped from elsewhere.

Contents by column (based on FRED codes - starting in 1967):
Montly: FEDFUNDS, UMCSENT, UNRATE, CPIAUCSL, INDPRO, HOUST, SP500
Montly spread: BAA, AAA, DGS10, DGS1
Quarterly: GDPC1


Reference:

Tino Berger, James Morley and Benjamin Wong, forthcoming, "Nowcasting the Output Gap", Journal of Econometrics.
